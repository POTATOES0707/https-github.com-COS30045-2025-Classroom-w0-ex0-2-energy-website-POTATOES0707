Television

Australia's television consumption has evolved significantly in recent years, driven by the rise of streaming services and digital platforms. Traditional TV viewing still holds a place, but many Australians now prefer on-demand content through services like Netflix, Stan, and Disney+.

This shift is influencing viewing habits, with audiences increasingly opting for flexibility in when and how they watch. Despite this, live TV, especially for sports and major events, remains popular. The growing trend towards high-definition and 4K viewing, coupled with the rise of smart TVs, is also shaping the future of television consumption in Australia.

Data Set Cleaning Process

This is the KNIME data cleaning and processing workflow. The process is divided into 2 parts: the left is the cleaning part and the right is the processing part which is responsible for the data visualization or charts.

First Step is reading the CSV file through a CSV reader node. The data is then filtered to remove any rows with missing values, ensuring that the analysis is based on complete data. Further more, the brand name is capitalized to ensure consistency in the data before going through a Column Filter node to remove any column that is unnecessary for the analysis. which in this case is the Grand Date column which is missing every values.

Second step is processing the data to create the charts. This is usually done by using group by node the group the column that is needed for the charts. And most of the time, there is a need for an extra node to calculate the data before it can be presented.
